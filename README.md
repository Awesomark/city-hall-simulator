# City Hall Simulator

A free, static browser prototype for a city-government policy simulator.

Open `index.html` in a browser to play locally. The same files can be published on GitHub Pages from the `main` branch.

Current prototype:

- Policy tabs for transportation, community safety, taxes, infrastructure, and housing
- Toggleable policies with income, expense, and approval effects
- City income, city expenses, approval, and annual balance
- Interest group pressure bars
- Monthly advancement
