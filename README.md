# City Hall Simulator

A free, static browser prototype for a city-government policy simulator.

Open `index.html` in a browser to play locally. The same files can be published on GitHub Pages from the `main` branch.

Current prototype:

- City name and population panel
- Group support bars with each group's population share
- City metrics for revenue, expenses, traffic, safety, housing, infrastructure, environment, trust, and approval
- Turn-based city questions with multiple answers
- Recent decision log
